Sample gem for Extending Integer
================================

Sample Usage
------------

    >> require 'fibonize'
    >> 156.closest_fibonacci
    => 144
    
Installation
------------

Build and install the gem
    $ git clone git://github.com/richardiux/fibonize.gem
    $ cd fibonize
    $ rake install
    
Development
--------------

Install development dependencies
    $ bundle

Running on irb console
    $ bundle exec irb

Running the tests
    $ rake
